Research Overview

Dr. Aditya Kumar Singh Pundir’s research focuses on the intersection of VLSI & Embedded Systems, Reconfigurable Hardware, and Artificial Intelligence. His work addresses reliability, fault tolerance, testing, and intelligent optimization in modern electronic systems, with applications spanning memory testing, CAD algorithms, and signal processing.

Research Interests & Domains

VLSI & Reconfigurable Hardware

  • Fault Tolerant Reconfigurable Hardware Architectures
  • SRAM-based FPGA Reliability
  • Design for Testability (DFT)
  • Hardware Redundancy and Repair Techniques

Built-In Self-Test (BIST)

  • Memory BIST Architectures
  • Embedded System Based Testing
  • On-chip Fault Diagnosis
  • Low Power Test Techniques

Artificial Intelligence & Machine Learning

  • AI-assisted Memory Testing and Repair
  • Machine Learning for Fault Prediction
  • Data-driven Hardware Reliability Analysis
  • Intelligent Test Pattern Generation

CAD Algorithms

  • Optimization Algorithms for VLSI CAD
  • Design Automation Techniques
  • Algorithmic Complexity Reduction
  • EDA Tool Optimization

Signal Processing

  • Digital Signal Processing Algorithms
  • Signal Analysis for Embedded Systems
  • Hardware Implementation of DSP Algorithms
  • AI-enabled Signal Processing Applications

Future Research Directions

Current and future research directions include the integration of Artificial Intelligence with hardware testing and repair, development of smart BIST frameworks, and exploration of energy-efficient fault tolerant systems for next-generation embedded and reconfigurable platforms.
